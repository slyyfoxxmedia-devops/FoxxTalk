# SlyyFoxx Media

A minimal full-stack application with Vite/React frontend and FastAPI backend.

## Features
- Landing page
- Blog content page
- Admin panel for creating posts
- Black background with burnt orange theme
- Orbitron font

## Setup
```bash
./setup.sh
```

## Running

### Backend
```bash
cd backend
source venv/bin/activate
python main.py
```

### Frontend
```bash
cd frontend
npm run dev
```

## URLs
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- Domain: slyyfoxxmedia.com
